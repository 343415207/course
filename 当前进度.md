## 📱 下一步操作

### 1. 测试小程序（推荐）

1. **打开微信开发者工具**
   - 导入项目：选择 `miniprogram` 目录
   - 填写 AppID（可使用测试号）

2. **配置开发环境**
   - 设置 → 项目设置 → 本地设置
   - 勾选"不校验合法域名"

3. **测试功能**
   - 登录功能
   - 首页 Banner 和课程列表
   - 课程详情和记录管理
   - 图片上传

### 2. 添加 TabBar 图标

在 `miniprogram/images/` 目录下添加：
- `home.png` / `home-active.png`
- `profile.png` / `profile-active.png`

### 3. 配置微信小程序

1. 在微信公众平台获取 AppID 和 Secret
2. 更新 `.env` 文件中的配置
3. 配置服务器域名（生产环境）

### 4. 真机调试

1. 获取本机局域网 IP
2. 修改 `app.js` 中的 `apiBaseUrl`
3. 在微信开发者工具中点击"预览"
