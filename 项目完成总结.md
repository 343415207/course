## 📱 使用指南

### 1. 启动服务
```bash
docker-compose up -d
```

### 2. 创建测试数据（可选）
```bash
docker-compose exec api node scripts/init-data.js
```

### 3. 测试小程序
1. 打开微信开发者工具
2. 导入 `miniprogram` 目录
3. 配置开发环境（不校验合法域名）
4. 开始测试

### 4. 访问管理界面
- phpMyAdmin: http://localhost:9080
- API 健康检查: http://localhost:9000/health

## 📝 注意事项

### 开发环境
- 小程序开发时需要设置"不校验合法域名"
- 真机调试时需要使用局域网 IP 或内网穿透
- TabBar 图标需要自行添加（已简化配置，可暂时使用文字）

### 生产环境
- 需要配置微信小程序 AppID 和 Secret
- 需要配置服务器域名和 HTTPS
- 需要配置腾讯云 COS 凭证
- 需要修改 JWT Secret 等敏感信息
