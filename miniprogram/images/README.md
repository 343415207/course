# TabBar 图标说明

## 需要的图标文件

请在此目录下添加以下图标文件：

1. **home.png** - 首页图标（未选中状态）
2. **home-active.png** - 首页图标（选中状态）
3. **profile.png** - 我的图标（未选中状态）
4. **profile-active.png** - 我的图标（选中状态）

## 图标规格

- **尺寸**: 81px × 81px
- **格式**: PNG（支持透明背景）
- **颜色**: 
  - 未选中：灰色 (#7A7E83)
  - 选中：绿色 (#3cc51f)

## 临时方案

如果暂时没有图标，可以：
1. 使用在线图标生成工具
2. 使用简单的文字图标
3. 暂时注释掉 tabBar 配置中的 iconPath

## 图标资源推荐

- [IconFont](https://www.iconfont.cn/)
- [Flaticon](https://www.flaticon.com/)
- [Icons8](https://icons8.com/)
